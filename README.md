Proyek ini merupakan implementasi sistem monitoring suhu dan kelembapan berbasis ESP32 yang terintegrasi dengan sensor DHT dan layar OLED, serta dilengkapi fitur webserver yang dapat diakses melalui Mode Access Point (AP) maupun Mode Station (STA). Sistem ini dirancang untuk menampilkan data suhu dan kelembapan secara real-time baik secara lokal melalui layar OLED maupun secara jarak jauh melalui browser pada perangkat seperti smartphone, laptop, atau komputer.

Pada Mode Access Point (AP), ESP32 berfungsi sebagai pemancar Wi-Fi mandiri sehingga pengguna dapat langsung terhubung ke jaringan yang dibuat oleh ESP32 tanpa memerlukan router. Mode ini sangat cocok digunakan di area yang tidak memiliki jaringan internet atau untuk keperluan instalasi awal. Sementara itu, pada Mode Station (STA), ESP32 terhubung ke jaringan Wi-Fi yang sudah ada (misalnya hotspot atau router sekolah), sehingga data sensor dapat diakses dari berbagai perangkat dalam jaringan yang sama.

Sensor DHT11/DHT22 digunakan untuk mengukur suhu dan kelembapan lingkungan, kemudian data tersebut diproses oleh ESP32 dan ditampilkan pada layar OLED dalam bentuk teks yang jelas dan ringkas. Selain itu, ESP32 menjalankan webserver internal yang menyajikan halaman web sederhana berisi informasi suhu, kelembapan, dan status koneksi. Halaman ini dapat diakses melalui alamat IP ESP32 menggunakan browser.

Proyek ini sangat relevan untuk pembelajaran IoT (Internet of Things), karena menggabungkan konsep sensor, mikrokontroler, jaringan Wi-Fi, dan webserver dalam satu sistem terintegrasi. Selain itu, proyek ini dapat dikembangkan lebih lanjut untuk aplikasi nyata seperti monitoring ruang kelas, laboratorium, gudang, atau rumah pintar. Dengan adanya dua mode koneksi (AP dan Station), sistem menjadi lebih fleksibel dan mudah diterapkan di berbagai kondisi jaringan.
