# ESP32_DHT_Oled_Webserver_AP_Client
Proyek ESP32 DHT OLED Webserver Mode AP dan Mode Station menampilkan data suhu dan kelembapan dari sensor DHT pada layar OLED serta melalui web server. Sistem dapat bekerja sebagai Access Point maupun terhubung ke WiFi, sehingga monitoring dapat dilakukan secara lokal dan jarak jauh.
